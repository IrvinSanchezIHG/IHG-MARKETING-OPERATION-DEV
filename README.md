# IHG Marketing Operations Repository

## Welcome

Welcome to the IHG Marketing Operations repository.

This repository serves as the central location for documentation, templates, scripts, technical assets, and operational resources used by the Marketing Operations team in day-to-day activities.

Here you will find information related to:

- Campaign development resources
- Salesforce Marketing Cloud assets
- Email templates and code snippets
- SQL queries
- Automation scripts
- Technical documentation
- Process documentation
- Best practices and standards
- QA and testing resources
- Project-specific files
- Reference materials and knowledge base

The purpose of this repository is to maintain version control, facilitate collaboration, preserve historical changes, and provide a single source of truth for the team.

Please follow the established Git workflow and repository structure when contributing new content.
